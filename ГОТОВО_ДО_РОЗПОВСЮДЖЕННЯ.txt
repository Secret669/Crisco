📦 ВАРІАНТ 1: PORTABLE (РЕКОМЕНДУЄТЬСЯ - ВЖЕ ГОТОВО!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Найпростіший спосіб - ВСЕ ВЖЕ ГОТОВО:

1️⃣ Створіть ZIP архів:
   • Клацніть правою кнопкою на папку "Crisco_Portable"
   • Виберіть "Відправити" → "Стиснута ZIP-папка"
   • Назвіть: Crisco_v1.0_Portable.zip

2️⃣ Розповсюдьте ZIP файл:
   • Електронною поштою
   • Через хмарне сховище (Google Drive, Dropbox)
   • На USB флешці
   • Через локальну мережу

3️⃣ Інструкція для користувачів:
   • Розпакувати ZIP
   • Запустити Crisco_Optimized.exe
   • Готово!

✨ ПЕРЕВАГИ:
   ✔️ Найпростіше - просто скопіювати папку
   ✔️ Не потребує прав адміністратора
   ✔️ Можна запускати з USB флешки
   ✔️ Легко оновлювати - просто замініть EXE

📦 ВАРІАНТ 2: ПРОФЕСІЙНИЙ ІНСТАЛЯТОР
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Для створення справжнього Windows інсталятора:

1️⃣ Встановіть Inno Setup (якщо ще не встановлено):
   🌐 https://jrsoftware.org/isdl.php
   
2️⃣ Відкрийте Inno Setup Compiler

3️⃣ Відкрийте файл: installer_setup.iss

4️⃣ Натисніть: Build → Compile (або F9)

5️⃣ Інсталятор буде створено:
   📂 installer_output\Crisco_Setup.exe

✨ ПЕРЕВАГИ:
   ✔️ Професійний вигляд
   ✔️ Автоматичне створення ярликів
   ✔️ Можливість деінсталяції
   ✔️ Встановлення в Program Files

⚠️ ВАЖЛИВА ІНФОРМАЦІЯ ДЛЯ КОРИСТУВАЧІВ:
────────────────────────────────────────────────────────────

На цільовому комп'ютері потрібно:

1. Windows 7/8/10/11 (64-біт)
2. Microsoft Access Database Engine
   (якщо не встановлено, інструкції в README.txt)

Програма НЕ потребує:
✗ Встановлення Python
✗ Встановлення додаткових бібліотек
✗ Прав адміністратора (для portable версії)
✗ Інтернет підключення

📊 ТЕХНІЧНА ІНФОРМАЦІЯ:
────────────────────────────────────────────────────────────

Розмір файлів:
• EXE файл: 28 МБ
• База даних: 647 КБ
• Загальний розмір Portable: ~29 МБ
• ZIP архів: ~10-15 МБ (стиснутий)

Що включено в EXE:
• Python 3.13 runtime
• Tkinter (GUI)
• tkcalendar (календар)
• pyodbc (з'єднання з Access)
• python-docx (створення Word документів)
• Всі ваші модулі (main.py, db_manager.py, тощо)
