  CRISCO - ПРОГРАМА ДЛЯ ВЕДЕННЯ ЗАМІН (Очищена версія)
═══════════════════════════════════════════════════════════

Ця папка містить лише необхідні файли для роботи програми.

ЗМІСТ ПАПКИ:
────────────────────────────────────────────────────────────

📄 ОСНОВНІ ФАЙЛИ ПРОГРАМИ:
  • main.py - головний файл програми
  • db_manager.py - менеджер бази даних
  • replacement_form.py - форма для створення замін
  • final_handbook_fix.py - форма довідника

📊 БАЗА ДАНИХ:
  • dataBase.mdb - база даних Access з інформацією про викладачів, 
    групи, дисципліни, аудиторії та відділення

📁 ПАПКИ:
  • Заміни/ - папка для збереження створених файлів замін
    (організовані за навчальними роками та місяцями)

📋 КОНФІГУРАЦІЙНІ ФАЙЛИ:
  • requirements.txt - список необхідних Python бібліотек
  • Crisco_Optimized.spec - файл конфігурації для PyInstaller

════════════════════════════════════════════════════════════

🚀 ЯК ЗАПУСТИТИ ПРОГРАМУ:

1. Встановіть Python 3.8 або новішу версію
2. Встановіть необхідні бібліотеки:
   pip install -r requirements.txt
3. Запустіть програму:
   python main.py

════════════════════════════════════════════════════════════

📦 ЯК СТВОРИТИ ВИКОНУВАНИЙ ФАЙЛ (EXE):

1. Встановіть PyInstaller:
   pip install pyinstaller
2. Створіть EXE файл:
   pyinstaller Crisco_Optimized.spec

Виконуваний файл буде створено в папці dist/
