📋 СИСТЕМНІ ВИМОГИ:
────────────────────────────────────────────────────────────
• Windows 7 / 8 / 10 / 11 (64-біт)
• 100 МБ вільного місця на диску
• Microsoft Access Database Engine (інструкції нижче)

════════════════════════════════════════════════════════════

💾 ВСТАНОВЛЕННЯ ТА ЗАПУСК:
────────────────────────────────────────────────────────────

1️⃣ Розпакуйте архів в будь-яку папку на вашому комп'ютері

2️⃣ Переконайтесь, що у папці є два файли:
   • Crisco_Optimized.exe
   • dataBase.mdb

3️⃣ Запустіть файл: Crisco_Optimized.exe

⚠️ ВАЖЛИВО: НЕ видаляйте файл dataBase.mdb!
   Він містить всю інформацію про викладачів та дисципліни.

════════════════════════════════════════════════════════════

⚠️ ЯКЩО ПРОГРАМА НЕ ЗАПУСКАЄТЬСЯ
────────────────────────────────────────────────────────────

Найімовірніше потрібно встановити Microsoft Access Database Engine:

1️⃣ Завантажте драйвер з офіційного сайту Microsoft:
   🌐 https://www.microsoft.com/en-us/download/details.aspx?id=54920

2️⃣ Виберіть правильну версію:
   • Для 64-біт Windows: AccessDatabaseEngine_X64.exe
   • Для 32-біт Windows: AccessDatabaseEngine.exe
   
   Як дізнатись розрядність Windows:
   Правий клік на "Цей комп'ютер" → Властивості → Тип системи

3️⃣ Запустіть завантажений файл та встановіть драйвер

4️⃣ Перезапустіть програму Crisco

════════════════════════════════════════════════════════════

🎯 ПОЧАТОК РОБОТИ:
────────────────────────────────────────────────────────────

Після запуску програми:

1. При першому запуску автоматично створюється папка "Zaminy"
   для збереження документів про заміни

2. Створіть новий навчальний рік:
   • Натисніть кнопку "Новий навч. рік" (праворуч)
   • Введіть рік (наприклад: 2025-2026)

3. Оновіть дані з бази:
   • Натисніть кнопку "Оновити дані"
   • Це завантажить список викладачів та дисциплін

4. Заповніть бланк заміни:
   • Натисніть "Бланк замін"
   • Заповніть необхідні поля
   • Збережіть документ

════════════════════════════════════════════════════════════

📁 СТРУКТУРА ФАЙЛІВ:
────────────────────────────────────────────────────────────

Після встановлення у вас буде наступна структура:

📂 Crisco/ (ваша папка з програмою)
├── 📄 Crisco_Optimized.exe    ← Основна програма
├── 📊 dataBase.mdb            ← База даних (НЕ видаляти!)
├── 📄 README.txt              ← Ця інструкція
└── 📁 Zaminy/                 ← Папка для документів (створюється автоматично)
    └── 📁 2025-2026/          ← Навчальні роки
        ├── 📁 Вересень/
        ├── 📁 Жовтень/
        └── ... (інші місяці)

💡 КОРИСНІ ПОРАДИ:
────────────────────────────────────────────────────────────

✅ Регулярно робіть резервні копії:
   • Скопіюйте файл dataBase.mdb в безпечне місце
   • Скопіюйте папку Zaminy з вашими документами

✅ Автозаповнення:
   • При введенні прізвища викладача почніть друкувати
   • З'явиться список для швидкого вибору

✅ Організація файлів:
   • Програма автоматично сортує файли за роками та місяцями
   • Використовуйте слайдер (кнопка "<" праворуч) для перегляду

✅ Друк документів:
   • Після створення бланку заміни його можна роздрукувати
   • Файли зберігаються у форматі Word (.docx)

════════════════════════════════════════════════════════════

🔧 ВИРІШЕННЯ ПРОБЛЕМ:
────────────────────────────────────────────────────────────

❌ Помилка: "Не вдалося підключитися до бази даних"
   → Перевірте, чи файл dataBase.mdb в тій же папці
   → Встановіть Microsoft Access Database Engine (див. вище)

❌ Помилка: "Не знайдено таблицю викладачів"
   → Натисніть кнопку "Оновити дані"
   → Перезапустіть програму

❌ Програма не запускається взагалі
   → Запустіть від імені адміністратора (ПКМ → "Запустити від імені адміністратора")
   → Перевірте антивірус - можливо він блокує програму
   → Встановіть Microsoft Access Database Engine

❌ Не зберігаються файли замін
   → Перевірте права доступу до папки
   → Створіть папку "Zaminy" вручну

❌ Крякозябри замість українських букв
   → Це відбувається тільки в консолі (якщо запускаєте з cmd)
   → В інтерфейсі програми все буде коректно
